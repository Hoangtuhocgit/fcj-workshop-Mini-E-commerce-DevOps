# Summary Report: "FCAJ GAMEDAY"

### Event Information

| Item | Details |
| --- | --- |
| Event name | FCAJ GAMEDAY |
| Date & time | 09:00, June 20, 2026 |
| Location | 26th Floor, Bitexco Tower, 02 Hai Trieu Street, Saigon Ward, Ho Chi Minh City |
| Role | Attendee |
| Format | Team-based multiple-choice quiz |
| Number of teams | 8 teams |

### Event Objectives

FCAJ GAMEDAY is a knowledge competition organized by the First Cloud Journey (FCAJ) community, testing understanding of AWS Cloud and Solutions Architect (SA) knowledge. Based on my attendance, the event pursued these objectives:

1. **Assess AWS knowledge:** Test mastery of AWS services, concepts, and best practices.
2. **Cloud and SA certification preparation:** Multiple-choice questions aligned with AWS Cloud and Solutions Architect content — helping members prepare for AWS Cloud and SA certification paths.
3. **Team competition:** 8 teams compete, encouraging discussion and consensus on answers — building teamwork skills.
4. **Community connection:** A platform for FCAJ members passionate about Cloud to learn from each other.

### Competition Format

| Item | Description |
| --- | --- |
| Teams | 8 groups participated |
| Format | Team-based multiple-choice question answering |
| Topics | AWS Cloud services, Cloud service usage skills, and Solutions Architect (SA) knowledge |
| Method | Each team discusses and selects answers together |

### Quiz Content

The competition focused on **multiple-choice questions** aligned with AWS Cloud services, Cloud service usage skills, and SA knowledge, covering Cloud and Solutions Architect scope:

| Area | Common Knowledge Topics |
| --- | --- |
| Architecture & Design | Highly available, fault tolerant, scalable solutions on AWS |
| Compute | EC2, Lambda, ECS, Fargate, Auto Scaling — appropriate service selection |
| Storage | S3, EBS, EFS, Glacier — use cases and features |
| Database | RDS, DynamoDB, Aurora, ElastiCache — comparison and application |
| Networking | VPC, Subnet, Route Table, NAT Gateway, VPN, Direct Connect |
| Security | IAM, Security Group, NACL, KMS, encryption, least privilege |
| Monitoring | CloudWatch, CloudTrail, X-Ray — monitoring and audit |
| Integration & Messaging | SQS, SNS, EventBridge, API Gateway |
| Cost & Optimization | Reserved Instance, Savings Plans, right-sizing, cost optimization |
| Disaster Recovery | Backup, RTO/RPO, multi-AZ, multi-Region strategies |

### Key Takeaways

**1. AWS Cloud & SA knowledge**

- Through the quiz questions, I reviewed AWS services from Cloud Practitioner and Solutions Architect perspectives — knowing when to use which service.
- Stronger grasp of high availability, scalability, fault tolerance, and disaster recovery concepts.
- Deeper understanding of IAM, VPC, and security — common quiz topics in SA exams.

**2. Question analysis skills** *(from observing quiz questions)*

- Through the quiz questions, I learned to read carefully, identify keywords (cost optimization, least downtime, serverless…) to choose correct answers.
- Eliminate wrong answers using foundational knowledge instead of guessing.
- SA questions often test **trade-offs** between solutions, not just technically correct single answers.

**3. Teamwork skills** *(observed from competing teams)*

- Group discussion adds perspectives — I saw members strong in different areas (network, database, security…) contribute to shared answers.
- How to explain answer choices and persuade teammates when opinions differ.
- Reaching consensus quickly under time pressure — important in real work environments.

**4. AWS certification direction**

- FCAJ GAMEDAY serves as **free practice** in a format close to AWS Certified Cloud Practitioner and Solutions Architect exams.
- Clearer identification of knowledge gaps to strengthen before taking Cloud and SA certifications.

### Event Experience

I attended FCAJ GAMEDAY at Bitexco Floor 26 as an **attendee**. Unlike Community Day (listening to talks), this was a **multiple-choice quiz competition** — **8 teams** competed to test AWS Cloud and SA knowledge.

**Competition atmosphere**

The hall had 8 teams seated together — tense yet energetic. When facing difficult questions, the teams discussed together, creating active learning even during competition.

**Impressions**

- Diverse questions covering many AWS services — not focused on one area only.
- Some **cost and architecture optimization** questions were challenging, requiring real SA thinking.
- Observing the competing teams helped me learn from how members discussed and shared knowledge during the quiz.

**Lessons learned**

FCAJ GAMEDAY is an effective and engaging way to review AWS. Team-based multiple-choice competition tests individual knowledge while building **technical discussion and shared decision-making** — essential skills for Cloud Engineers in real work environments.
